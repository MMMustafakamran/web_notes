REACT
React: A Front-end Development Library
React, or ReactJs is an open-source JavaScript library used to develop single-page
web applications. It’s one of the popular libraries to build UI (front-end) for web and
mobile apps. It was developed and backed by Facebook in 2011 and has gained
popularity since then.
